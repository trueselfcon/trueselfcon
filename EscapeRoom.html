Event Type: In-Person, RSVP Event Cost: $50/Person Cosplay: Optional/Encouraged

Welcome everyone! As the above suggests, you are being invited to a custom-made escape room based around the Persona 4 Golden game, where you will find your memory and patience tested. This event is being set up not only so everyone can get together outside of the convention scene for some fun but also as a way to continue raising funds for an eventual Fan-based Persona convention. Below we have tried to answer all the questions we can think might be asked, but you are welcome to mail ryoma_meeiz@yahoo.com or contact Ryoma on Discord under ryokurama if you don't see things answered here.


Please be aware that the escape room portion itself will be approximately 90 minutes, the rest will be games and mingling.
